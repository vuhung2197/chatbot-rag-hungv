# Tài liệu các Module Backend

Thư mục này chứa tài liệu kỹ thuật chi tiết cho các module backend của Ứng dụng English Chatbot.

## Các Module

| Module | Mô tả | Service chính |
|---|---|---|
| [Auth](./auth.md) | Xác thực người dùng, phiên đăng nhập và OAuth | `AuthService` |
| [Chat](./chat.md) | Lõi logic RAG, phân loại ý định và streaming | `ChatService` |
| [Knowledge](./knowledge.md) | Quản lý cơ sở tri thức và phân mảnh tài liệu | `KnowledgeService` |
| [Subscription](./subscription.md) | Các gói đăng ký, thanh toán và nâng cấp | `SubscriptionService` |
| [Upload](./upload.md) | Tiếp nhận và xử lý file (.txt, .docx) | `UploadService` |
| [Usage](./usage.md) | Theo dõi hoạt động người dùng và thực thi giới hạn | `UsageService` |
| [User](./user.md) | Quản lý hồ sơ và ảnh đại diện | `UserService` |
| [Wallet](./wallet.md) | Hệ thống ví tài chính và giao dịch | `WalletService` |
